For every public report we collect, a walk-forward model predicts each release from prior data only (no look-ahead) and is scored against the naive persistence baseline. NOWCAST = a genuine pre-release prediction (the report's data-date precedes its publication, like COT). FORECAST = a short-horizon next-value forecast of near-real-time data (no release gap — labeled honestly, not sold as a scoop). MASE < 1 beats persistence; directional accuracy = share of releases whose direction we called right. This is a deliberately simple generic univariate model — the COT nowcast (richer features) is the reference build.
